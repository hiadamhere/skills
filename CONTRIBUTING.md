# Contributing

Thanks for contributing! We welcome help in expanding and refining master-level AI agent skills.

## How to Contribute

1. **Discuss first:** Search the existing issues or open a new one to discuss changes or skill proposals before writing code.
2. **Review Guidelines:** Ensure any new skill or update aligns with our documentation standards defined in `CLAUDE.md`.
3. **Submit a PR:** Keep commits clean and link your PR to the approved issue.

## Pull Request Verification

Every submission is run through our automated CI/CD pipeline checks. These tests verify the repository formatting, check for broken links, and validate our ground-truth reference mappings where applicable (such as C# library API signatures).
